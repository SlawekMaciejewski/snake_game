# SNAKE GAME
A simple SNAKE game using GUI from Turtle graphics. 
- The code was written to use documentations https://docs.python.org/3/library/turtle.html
- In this case, I was learning animation and OOP.

############### Keys #####################
- "W" -> Forwards
- "S" -> Backwards
- "A" -> Counter-Clockwise
- "D" -> Clockwise
- "C" -> Clear
